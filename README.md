# Competitive Matchmaking ( FaceIt clone lol )
Competitive Matchmaker is a web-based matchmaking platform created by esdeekido for the Roblox tactical shooters Counter Blox and Blox-Strike. It provides a competitive environment where players can register, queue for matches, track their competitive history, and participate in organized leagues.

Designed for the Counter-Blox Premier League (CAPL) and Blox-Strike Premier League (BAPL) communities, the platform streamlines matchmaking, player management, and league operations.

## Purpose
Competitive Matchmaker was built to provide a centralized platform for competitive Roblox FPS communities. It simplifies player matchmaking, league administration, and record keeping while giving players an organized experience for ranked matches, tournaments, and seasonal competitions across both Counter Blox and Blox-Strike.
